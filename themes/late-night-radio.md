# 深夜电台 Late Night Radio

一场独坐深夜的听觉仪式。配色灵感来自真空管放大器微光下的暗室——底色是接近黎明前最深的夜空蓝黑，主色调是电子管灯丝发出的暖橙色辉光，辅以旧收音机拉丝铝面板的青灰调。温暖但不甜腻，深沉但不死寂。

## 用途

桌面电台应用，暗色主题 UI，音乐播放器，深夜场景应用。

## 字体

- **标题 / 台名**：Noto Serif SC（思源宋体）——旧式电台的典雅衬线感
- **正文 / 界面**：DM Sans ——温暖干净的几何无衬线
- **频率 / 数据**：Fira Code ——等宽，模拟收音机频率显示

## 纹理

- 全局 3% 噪点叠加（SVG feTurbulence），消解纯色的数码感
- 侧边栏：细微渐变 + backdrop-filter 模糊，模拟磨砂玻璃
- 选中卡片：暖橙色外发光 + 微弱的渐变边框
