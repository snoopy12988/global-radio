# 全球电台 Global Radio

基于 Electron 的桌面全球音乐电台收听程序。

## 功能

- 🌍 按国家/地区浏览全球电台
- 🎵 按音乐流派筛选
- 🔍 电台搜索
- ❤️ 收藏夹
- 🕐 播放历史
- 🔊 正在播放 - 动态音柱频谱可视化
- 🚫 播放失败自动隐藏
- 🔄 启动自动续播

## 技术栈

- Electron 33
- Radio Browser API
- HLS.js（m3u8 流支持）
- Web Audio API（实时频谱）

## 启动

```bash
npm install
npm start
```
